General Typography

Font Type

font-family: comic sans ms

Headings

h1 Element

The h1 element are to be used as the page header only. They should only be used once per page. They should be centered within the page. The size should be the default size. Check the title at top of the page for an example.

h2 Element

The h2 element should be used as the headers for main sections. This should also be centered using "text-align: center;". The size should be default size. Check the -General Typography- title located underneath the page title at the top of the page for an example.

h3 Element

The h3 element should be used for all remaining header types that are not included in the h1 and h2 element guidelines. The size should remain at default size. Check the section title--h3 Element-- for an example.
